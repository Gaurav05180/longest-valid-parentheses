# Longest Valid Parentheses

## Problem Statement

Given a string containing only '(' and ')', return the length of the longest valid (well-formed) parentheses substring.

---

## Language

Java

---

## Solutions Included

| Solution            | Time  | Space |
|---------------------|------ |-------|
| Brute Force         | O(n³) | O(n)  |
| Dynamic Programming | O(n)  | O(n)  |
| Stack               | O(n)  | O(n)  |
| Two Pass Traversal  | O(n)  | O(1)  |
